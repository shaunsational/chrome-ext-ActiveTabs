Keyboard Shortcuts  |  Tips  |  Keywords

ENTER jump to the highlighted tab
ARROW UP select previous tab, will loop to end
ARROW DOWN select next tab, will loop to beginning
S jump to the search box, can minimize tab results this way
X close overlays, esc seems to be reserved
C close the highlighted tab, will prompt for confirmation
M mute/unmute the highlighted tab
P pin/unpin the highlighted tab
? will open/close this popup
  1. Using the search box you can quickly narrow your tab list down.
  2. Searching for keywords can help narrow it further
_audio will show all tabs currently playing audio
_muted will show all muted tabs
_pinned will show all pinned tabs
_highlighted will show all highlighted tabs
_incognito will show all incognito tabs
NOTE: searching for keywords will also show any tab which meets the search criteria, its not perfect, click a keyword for an example.
